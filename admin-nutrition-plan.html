Plano alimentar do aluno

Este sistema apenas exibe o plano cadastrado manualmente pelo profissional. Ele não gera prescrição nutricional automática.

Escreva o plano como texto normal. Cada título vira uma refeição e cada linha com hífen vira um item. Cada título vira um bloco de substituição. Os itens devem ficar em linhas com hífen. Escreva uma regra por linha.

Informe token e email para carregar ou salvar o plano.
