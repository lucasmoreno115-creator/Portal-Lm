# Operations Guide — LM Premium 3.0.0

## Aluno Premium

1. Confirmar cadastro e identidade Premium ativa.
2. Acompanhar status de consulta.
3. Verificar pendências abertas.
4. Conferir plano alimentar publicado e feedback semanal disponível.

## Anamnese

1. Enviar link de anamnese quando aluno estiver apto.
2. Validar recebimento.
3. Analisar conteúdo no fluxo administrativo.
4. Registrar decisão ou pendência quando necessário.

## Feedback

1. Disponibilizar feedback semanal no período operacional.
2. Monitorar submissões ausentes.
3. Preparar lembretes quando aplicável.
4. Analisar feedback recebido.
5. Registrar decisão profissional.

## Plano

1. Consultar plano atual publicado.
2. Criar rascunho quando houver necessidade operacional.
3. Atualizar rascunho.
4. Publicar somente após revisão profissional.
5. Arquivar ou duplicar plano conforme histórico.

## Workspace

1. Usar resumo para triagem diária.
2. Filtrar alunos por status e pendências.
3. Buscar aluno por nome/e-mail.
4. Abrir contexto do aluno para tomada de decisão.

## Prontuário

1. Revisar histórico longitudinal.
2. Registrar follow-up quando houver ação relevante.
3. Relacionar decisões a feedbacks, planos ou pendências quando aplicável.
4. Evitar duplicar decisões já registradas.

## Saturday Review

1. Abrir resumo de sábado.
2. Revisar feedbacks pendentes.
3. Priorizar alunos com pendências abertas.
4. Registrar decisões e próximos passos.
